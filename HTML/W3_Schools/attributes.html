HTML Attributes

Attributes provide additional information about HTML elements.
They comprise of the "href" attribute and the "src" attribute.

(1). The "href" Attribute.

The "a" tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to.
The format would be {a href="https://twitter.com/home"}, with the angle brackets at the beginning and the end.
An example could be Link to W3schools
Second example My Twitter

(2). The "src" Attribute

The "img" tag is used to embed an Image to a HTML page.
The "src" attribute specifies the path to the image to be displayed.
The format would be {img src="img_girl.jpg"}, with angle brackets at the beginning and the end.

There are two ways to specify the URL in the src attribute:
(a)Absolute URL
This is are Links to an external image that is hosted on another website.
The only disadvantage with images from external links is that you cannot control them as they can be changed or removed.
The format example could be like :{src="https://www.w3schools.com/images/img_girl.jpg".}
Below is a display example of a laptop stickers image from twitter.



(b)Relative URL
This one links to an image that is hosted within the website. Here, the URL does not include the domain name.
Here, the URL does not include the domain name. If the URl begins without a forward slash, it will be relative to the current page but if it begins with a slash, it will be relative to the domain name.
The format without a slash would be {src="img_girl.jpg"}
The format with a slash would be {src="/images/img_girl.jpg"}.
Note: It is almost always best to use relative URLs (those without a slash). They will not break if you change domain.

(3). The "alt" Attributes

It comes after the src attribute in the img tag.
It specifies an alternate text for an image, if the image for some reason cannot be displayed.Failure of display could be caused by a slow connection, an error in the src attribute, or if the user uses a screen reader.
Its format is {img src="img_girl.jpg" width="500" height="600"} closed by the angle brackets.

(4). The "style" Attribute

The style attribute is used to add styles to an element, such as color, font, size, and more.
As an example, to change the color of text in a paragraph, immediately after the opening part of a paragraph ie {the first "p"}, inside it is where you include the style attribute to change color of the specified paragraph.
Format would be {style="color:"The desired color either red, blue or any other desired. ";"}

(4). The "lang" Attribute

You should always include the lang attribute inside the tag, to declare the language of the Web page.This is meant to assist search engines and browsers.
To specify English as the language, we use {lang="en"} inside the html tag.
