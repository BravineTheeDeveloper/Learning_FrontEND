HTML Tables

HTML tables are used to display data in a way that is easy to read.
HTML tables allow developer to arrange data into rows and columns.
A Table in HTML consists of tables rows and columns.

Each table cell is defined by a "td" and a "/td" tag, which has a full definition of Table data
Everything between the two tags are the contents of table cell.

Each table row starts with a "tr" and ends with a "/tr" tag. There the "tr" stands for table rows. The more the number of "tr" opening tags, the more the rows will there be in the specific table.
You can have as many rows as you like in a table; just make sure that the number of cells are the same in each row.

HTML Tables Borders

HTML tables can have borders. The borders are defined by the CSS border property of the td tag.
Tables can have borders of different types and shapes.
To avoid having double borders; you set the CSS border-collapse property to collapse. This will make the borders collapse into a single border.

It is also possible to achieve round table borders. With the border-radius property, the borders get rounded corners.
