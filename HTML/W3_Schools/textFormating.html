TEXT FORMATTING

HTML contains several elements for defining text with a special meaning.
Formatting elements were designed to display special types of texts like:


1. The bold text is enclosed by elements {b OR strong}
Examples would be:
Put your money where your mouth is!
This text was really supposed to be bold!
There is still a chance for my man!

2. The {i and em} elements.
The {i} element is used make text to be printed out as italic. Most commonly used to display a technical term or a foreign language.
Examples would be:
This text is Italic
Our numbers have risen!
Welcome home son!
Hello screen reader!
Note: The [em] element is a bot different from the {i} element as a screen reader will pronounce the words in {em} with emphasis and using verbal stress.

3. The {small} text.
This element is used to display smaller text ie.
This is some smaller text!
He will not win the election.
Why did he decide too get away like that!

4. The {mark} element
This element defines text that should be highlighted or marked.
Examples are:
Don't forget to buy Milk form the shop.
You are a gem!.
He went to steal Money from the bank!

5. The {del} element
This element defines text that has been deleted from a document. Browsers will usually strike a line through deleted text.
Examples:
My favorite color is blue red.
I will try visiting on Monday Tuesday.
i will come along with a book paper.

6. The {ins} element
It defines a text that has been inserted into a document. Browsers will usually underline inserted text.
Examples are:
My favorite color is RED
I am the LEADER of this organization.
let us get the PAPERWORK done.

7. The {sub} and the {sup} element
The {sup} element defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.
Examples could be:
The product of 22 is 4.
24 makes a perfect square.

On the other hand, the {sub} element defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.
They are mostly used in chemical formulas like Water. Examples can be:
H2O is the chemical formulae for Water.
CO2 is the chemical formulae for Carbon(iv) Oxide.

Chapter Summary

1. The {b/strong} tag defines bold text.
2. The {em} element defines emphasized text.
3. The {i/} defines part of text in an alternate voice or mood.
4. The {small} element defines smaller text.
5. The {sub/sup} elements defines text as either subscript or superscript.
6. The {ins} element defines inserted text which eventually is underlined.
7. The {del} element defines deleted text therefore a line will strike through the text.
8. The {mark} element defines marked/highlighted text.
