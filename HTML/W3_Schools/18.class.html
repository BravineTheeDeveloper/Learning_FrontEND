Multiple Classes
HTML elements can belong to more than one class.
When you have more than one class in a HTML element then then the element will get both CSS styles from the mentioned Classes.

Different HTML elements can share the same class ,i.e a heading and a paragraph.

Chapter Summary.

  • HTML class is used to add styling to the html element.
  • The HTML class attribute specifies one or more class names for an element
  • Classes are used by CSS and JavaScript to select and access specific elements.
  • The class attribute can be used on any HTML element
  • The class name is case sensitive.
  • Different HTML elements can point to the same class name
  • JavaScript can access elements with a specific class name with the getElementsByClassName() method.
