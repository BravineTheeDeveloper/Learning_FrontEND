HTML Images

The HTML {img} tag is used to embed an image in a web page.
This tag has two attributes in it;
- src: This specifies the path to te image.
- alt: This specifies an alternative text for the image.

To specify the height and the width of an image, you can use the style attribute inside the img tag.
An Example could be: style="width:500px;height:600px;
Alternatively, you can use the width and height attributes.
