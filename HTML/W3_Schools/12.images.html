HTML Images

The HTML {img} tag is used to embed an image in a web page.
This tag has two attributes in it;
- src: This specifies the path to te image.
- alt: This specifies an alternative text for the image.

To specify the height and the width of an image, you can use the style attribute inside the img tag.
An Example could be: style="width:500px;height:600px;
Alternatively, you can use the width and height attributes.

The width, height, and style attributes are all valid in HTML.
However, we suggest using the style attribute. This is because it prevents styles sheets from changing the size of images.

If you have your images in a sub-folder, you must include the folder name in the src attribute.

HTML also allows use of animated GIF's. When including it in the src attribute, you should end it with the extension {.gif}



To use an image as a link, put the img tag inside the a tag.

Another way of applying Images to a website is through Image Floating
Use the CSS float property to let the image float to the right or to the left of a text.
You can float an image either to the left or to the right of a paragraph with regard to your preferences.
