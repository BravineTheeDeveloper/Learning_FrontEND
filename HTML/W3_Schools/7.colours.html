HTML COLORS

HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, or HSLA values.
You can set the background color of a HTML page using the syntax {style="background-color:specified color;"}.
To change the color of a specific text ie, either Heading or Paragraph, you include the syntax {style="color:specified color;"}.
To change border colors in HTML, we use the syntax {style="border:2px solid preferred color;"}

HTML RGB and RGBA Colors

An RGB color value represents RED, GREEN, and BLUE light sources.
An RGBA color value is an extension of RGB with an Alpha channel (opacity).
In HTML, a color can be specified as an RGB value, using this formula: {rgb(red, green, blue)}
