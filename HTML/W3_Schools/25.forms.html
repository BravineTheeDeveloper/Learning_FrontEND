A drop down-list.

The Select element also has abilities to create drop down list.
It is used together with the Option element which defines an option that can be selected.
The Size element is used to specify the number of visible values.
The Multiple element is used to allow the user to select more than one option when they hold the Ctrl key on the keyboard.

Data List

The datalist element provides a lists of predefined options for an input element. When the list contains more than one option, it will display as drop-down menu.
To use this feature you need to add attribute "list" with datalists ID inside input tag. You also have to define all possible values that could appear on dropdown using < and >. Each item should end up with semicolon (;) character.
