HTML STYLES

Formats for styles!
The general syntax used for applying different styles in HTML is {tagname style="property:value;"} whereby the default tagname is "style".
To change the background color, we insert the syntax above inside the opening tag for the the boy element. ie, {body style="background-color:"desired color.;""}
To change text COLOR, you place the style syntax inside the opening tag for the paragraph or heading element. ie, {p style="color:tomato;"}
To change text FONT, you place the style syntax inside the opening tag for the paragraph or heading element. ie {p style="font:verdana;"}
To change text FONT-SIZE, you place the style syntax inside the opening tag for the paragraph or heading element. ie {p style="font-size:200% or any required;"}

Text Alignment
The text-align property defines the horizontal text alignment for an HTML element.
