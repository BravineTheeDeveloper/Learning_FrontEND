HTML Lists

This allows web developers to group a set of related items in lists.
An unordered list starts with the "ul" tag. Each list item starts with the "li" tag.
The list items will be marked with bullets (small black circles) by default because they are unordered lists.
Example:

  • Milk
  • Eggs
  • Bread
  • Coffee
An ordered list starts with the "ol" tag. Each list item starts with the "li" tag.
The list items will be marked with numbers by default because they are ordered lists.
Example:
  1. Tea
  2. Coffee
  3. Milk
  4. Eggs
The CSS list-style-type property is used to define the style of the list item marker. It can have one of the following values:
Value Description
Disc Sets the list item marker to a bullet (default)
Circle Sets the list ite marker to a Circle
Square Sets the list item maker to Square
None The list items will not be marked

Nested Lists in HTML

You can create nested lists in HTML where you include a list inside a list.
Example:

  • Coffee
  • Tea
    • Black Tea
    • Green Tea
  • Milk
